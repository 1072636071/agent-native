# Macros — 智能体指南

Macros 是一个 agent-native 语音和营养追踪应用。智能体通过
action 和 SQL 状态处理食物、餐食、卡路里/宏量素、语音纠正、
统计和导航。

## 核心规则

- 绝不硬编码 API 密钥、令牌、Webhook URL、签名密钥、私有 Builder/内部数据、客户数据或类似凭据的字面量。使用密钥/OAuth/运行时配置，示例中使用明显的占位符。
- 使用 action 处理餐食、食物、卡路里/宏量素更新、语音命令处理、
  统计和导航。不要直接变更应用表。
- 当来源未知时，不要编造营养值。询问、透明使用默认值或标记估计值。
- 语音转录可能包含常见的食物/名称错误；在进行破坏性变更前确认模糊条目。
- 当活跃的餐食、日期、食物或统计上下文不明确时，使用 `view-screen`。
- 保持健康/营养指导非医疗性，聚焦于追踪数据。

## 应用状态

- `navigation` 暴露当前日期、餐食、食物条目、统计和设置视图。
- `navigate` 将 UI 移动到日志、餐食、统计和设置。

## 技能

在更改卡路里/宏量素行为之前阅读 `update-calories`。使用 `actions`、
`storing-data`、`security`、`frontend-design` 和 `shadcn-ui` 进行框架
工作。
